# ColorSeeker

Android app built with Jetpack Compose.

## Quickstart

1. Clone repo:
   ```
   git clone https://github.com/airdropfarmer94-ai/colorseeker.git
   ```
2. Open in Android Studio.
3. Build & Run.

## Features

- Modern Android architecture (Jetpack Compose)
- Ready for feature development!

## Build

```sh
./gradlew build
```
